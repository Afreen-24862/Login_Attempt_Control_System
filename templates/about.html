About LoginGuard

A full-stack demonstration of brute-force protection and secure authentication engineering.

Project Overview

LoginGuard is a complete Login Attempt Control System built with Flask. It demonstrates how modern applications defend against brute-force attacks by tracking failed authentication attempts, locking accounts after repeated failures, and automatically restoring access once a cooldown period has passed. The platform includes a user dashboard, an administrator control panel, interactive analytics, and exportable audit reports — all wrapped in a polished, responsive cybersecurity-themed interface.

Key Features

  • Secure registration & login
  • Account lockout after repeated failures
  • Automatic unlock timer
  • Role-based access (user / admin)
  • Real-time analytics dashboards
  • CSV / PDF exportable reports

Security Features

  • Salted, hashed password storage
  • CSRF protection on all forms
  • Parameterized SQL (no injection)
  • Secure, HTTP-only session cookies
  • Rate limiting on login attempts
  • Structured security audit logging

Authentication Workflow

1 User submits login credentials
2 System checks if account is currently locked
3 Password verified against hashed value
4 Failure increments counter; 5 failures triggers lockout
5 Lockout auto-expires after the cooldown window
6 Success resets counter and starts a secure session

Technology Stack

Backend: Python 3.12, Flask, Flask-Login, Flask-WTF

Frontend: HTML5, CSS3, Bootstrap 5, JavaScript

Database: SQLite

Analytics: Plotly, pandas

Future Enhancements

  • Two-factor authentication (TOTP)
  • Email-based password reset
  • IP geolocation & anomaly detection
  • OAuth / SSO integration

Developer Information

Built as an educational and portfolio project demonstrating full-stack development and applied cybersecurity engineering practices.
